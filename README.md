\# E-Learning Platform



Dự án tốt nghiệp - Nền tảng học trực tuyến 



\## Tech Stack



\- \*\*Backend:\*\* Spring Boot 3.2, Java 17

\- \*\*Database:\*\* PostgreSQL 15

\- \*\*FrontEnd:\*\* React

\- \*\*Tools:\*\* Docker, Git



\## Setup Database (Docker)



\### Yêu cầu

\- Docker Desktop đã cài đặt



\### Khởi động database



```bash

\# 1. Clone project

git clone <repo-url>

cd elearning-platform



\# 2. Start database

docker-compose up -d



\# 3. Kiểm tra
